Tags semânticas

A tag <strong> significa que o termo delimitado possui força dentro da frase. Logo ele aparecerá em negrito.

Já a tag <em> significa que queremos dar ênfase ao termo. Logo, ele aparecerá em itálico.

Para marcar o texto, usamos a tag <mark>. Logo ao usa-la, o texto aparecerá marcado.

Para deixar o texto pequeno é usada a tag <small>.

A tag <del>, significa que o texto está ali, pode ser lido, mas deve ser considerado pelo leitor.

A tag <ins> serve para sublinhar o texto, e significa que o texto está ali, deve ser lido e você deve prestar atenção nele.

Para apresentar um texto sobrescrito, usamos a tag <sup>, (Ex. x 2). Já para representar um texto subscrito, usamos a tag <sub>, (Ex. H 2O).

Para representar o código fonte em alguma linguagem, usamos a tag <code>. Ex.: System.out.print("Olá, mundo");

Para incluir a formatação do texto como ele é digitado, usamos a tag <pre>.

Para citações em bloco, usamos a tag <blockquote>. Com isso o texto ganha um recuo. É possível colocar um link para a citação através do parâmetro cite Ex.:
